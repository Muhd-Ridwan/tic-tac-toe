# TIC TAC TOE

It's a project from learning react in Udemy. Then I add some other features such as swapping
Player 1 & Player 2 after game ends with draw or either player won.
Actually I dont plan to host it, but when I first bought my domain in cloudflare,
I got curious and can say addicted, to host many web pages, just to get familiar with it.

## Local Dev Setup

1. Copy URL and clone in your machine
2. Open terminal and install dependencies by typing ```npm install```
3. Run locally ```npm run dev```
